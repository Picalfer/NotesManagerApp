# Notes Manager App

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Architecture](https://img.shields.io/badge/Pattern-MVVM-blueviolet.svg)](https://developer.android.com/topic/architecture)

Полнофункциональное приложение для управления преступлениями с возможностью добавления фото, контактов и генерации отчетов.

## Функциональность

- Создание и редактирование записей о преступлениях
- Съемка и прикрепление фотографий
- Выбор подозреваемых из контактов телефона
- Генерация отчетов с деталями преступления
- Сохранение данных в JSON формате
- Поддержка планшетов и телефонов (мастер-деталь)

## Технологии

- Kotlin
- Fragments
- Camera API
- Contacts API
- FileProvider
- JSON Serialization
- ViewPager
- Permission handling
